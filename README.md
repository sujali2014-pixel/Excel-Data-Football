# Excel-Data-Football
This is a project assignment to display simple soccer statistics and soccer league data.
========================================================================================
Deskripsi Karya

Produk berbasis Microsoft Excel berupa aplikasi untuk menampilkan data statistik pertandingan sepak bola yang dirancang mencatat, mengelola, dan menganalisis data dengan cara lebih sistematis. Aplikasi ini dilengkapi dengan fitur input data, perhitungan otomatis, serta visualisasi dalam bentuk grafik dan dashboard sederhana, sehingga memudahkan pengguna dalam memahami isi data yang ada, menentukan tim sepak bola terbaik, dan lain sebagainya.
